STUB PAGE FOR VETRAULICH (Discord-Valorant interface)

Vetraulich is a dynamic interface between the Valorant API and discord
servers. It is intended for easy use for players to gather statistics
about their games and their friends as well as providing support for managing
custom matches and recording their results.

I also plan on adding extensive gameplay analysis features and potential tools
for managing small/community tournaments (play-by-plays, overlay information, etc.)
